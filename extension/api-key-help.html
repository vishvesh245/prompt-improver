API Key Setup

Get Your API Key

Follow the steps below for any provider. Takes ~2 minutes.

🟠

Anthropic (Claude)

Claude Haiku / Sonnet

  1. Go to console.anthropic.com

    Create an account if you don't have one — it's free to sign up.

  2. Click API Keys in the left sidebar.

  3. Click "Create Key" and give it a name (e.g., "PromptPulse").

  4. Copy the key and paste it into PromptPulse.

    The key is only shown once — save it somewhere safe!
✅

Your key should start with sk-ant-api03-...

💰 Cost estimate

PromptPulse uses Claude Haiku by default — one of the cheapest models. Each analysis costs roughly $0.001–$0.003 (less than a penny). You'd need to run ~300 analyses to spend $1.

🟢

OpenAI (ChatGPT)

GPT-4o mini

  1. Go to platform.openai.com/api-keys

    You need an OpenAI Platform account (separate from ChatGPT Plus).

  2. Click "Create new secret key".

  3. Give it a name (e.g., "PromptPulse") and click Create.

  4. Copy the key and paste it into PromptPulse.

    The key is only shown once — save it somewhere safe!
✅

Your key should start with sk-...

💰 Cost estimate

PromptPulse uses GPT-4o mini — very affordable. Each analysis costs roughly $0.001–$0.003. You'd need ~300+ analyses to spend $1.

🔵

Google (Gemini)

Gemini Flash

  1. Go to aistudio.google.com/app/apikey

    Sign in with your Google account.

  2. Click "Create API key".

  3. Select a Google Cloud project (or let it create one for you).

  4. Copy the key and paste it into PromptPulse.

✅

Your key should start with AIza...

💰 Cost estimate

Google offers a free tier for Gemini API (up to 60 requests/minute). Beyond that, Gemini Flash is extremely cheap — roughly $0.0001 per analysis.

🔒

Your key stays private

Your API key is stored locally in your browser and sent to our secure server over HTTPS for analysis. Your key is never stored on our servers — it is only used for the duration of each request.
