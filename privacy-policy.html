Privacy Policy

Prompt Improver — Chrome Extension · Last updated: March 14, 2026

We do not sell, share, or monetize your data. Your prompts are analyzed in real-time and never stored on our servers.

1. What This Extension Does

Prompt Improver analyzes the text you type into AI chat interfaces (ChatGPT, Claude, and Gemini) and provides a quality score with suggestions for improvement. The analysis is performed by an AI model via our secure backend server.

2. Data We Process

Data Purpose Stored?
Prompt text Sent to our server for AI-powered analysis Not stored. Processed in real-time and discarded immediately after the response is returned.
API key (if provided) Forwarded to the chosen AI provider (Anthropic, OpenAI, or Google) for authentication Stored locally in your browser only (chrome.storage.local). Transmitted over HTTPS to our server, which forwards it to the AI provider. Never stored server-side.
Free usage count Tracks how many free analyses you've used (limit: 3) Stored locally in your browser only.

3. How Data Flows

When you click the Prompt Improver badge or it analyzes your prompt:

  • Your prompt text is sent over HTTPS to our backend server (prompt-improver-production.up.railway.app).
  • Our server forwards the prompt to the AI provider (Anthropic, OpenAI, or Google) for analysis.
  • If you provided your own API key, it is included in the request to authenticate with the AI provider.
  • The AI provider's response (score, issues, improved prompt) is returned to your browser.
  • No data is logged, cached, or stored on our server at any point.

4. Data We Do NOT Collect

  • We do not collect personal information (name, email, etc.)
  • We do not track browsing history or page content beyond the text you type in AI chat inputs
  • We do not use cookies, analytics, or tracking pixels
  • We do not sell or share any data with third parties
  • We do not serve advertisements

5. Third-Party Services

Depending on your configuration, your prompt text may be processed by one of the following AI providers:

  • Anthropic — Privacy Policy
  • OpenAI — Privacy Policy
  • Google — Privacy Policy

Our backend server is hosted on Railway (Railway Privacy Policy).

7. Data Retention & Deletion

  • Server-side: No data is retained. Prompts and API keys are processed in memory and discarded immediately.
  • Browser-side: Your API key and usage count are stored in chrome.storage.local. To delete:
    • Click the extension icon → "Remove Key", or
    • Uninstall the extension (all local data is automatically removed)

8. Security

  • All data is transmitted over HTTPS (TLS encryption)
  • API keys are stored in Chrome's secure extension storage, isolated from web pages
  • The extension only runs on the 4 specific AI chat domains listed above
  • No remote code execution — all extension code is bundled locally

9. Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect data from children.

10. Changes to This Policy

If we make material changes, we will update the "Last updated" date at the top. Continued use of the extension after changes constitutes acceptance of the updated policy.
